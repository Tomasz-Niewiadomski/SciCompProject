Detailed Description

Ranges of characters in a regular expression.

Storing a range of characters in a regular expression e.g [a-z] or [^a-z] or [a-z0-9] or [^a-z0-9]. It is used to store the ranges in a special matcher. All Ast objects have a vector of ranges, but only the SpecialAst should use it.
