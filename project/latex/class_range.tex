\doxysubsection{Detailed Description}
Ranges of characters in a regular expression. 

Storing a range of characters in a regular expression e.\+g \mbox{[}a-\/z\mbox{]} or \mbox{[}$^\wedge$a-\/z\mbox{]} or \mbox{[}a-\/z0-\/9\mbox{]} or \mbox{[}$^\wedge$a-\/z0-\/9\mbox{]}. It is used to store the ranges in a special matcher. All \mbox{\hyperlink{class_ast}{Ast}} objects have a vector of ranges, but only the \mbox{\hyperlink{class_special_ast}{Special\+Ast}} should use it.
