\doxysubsection{Detailed Description}
Alternatives (the \textquotesingle{}or\textquotesingle{} operation) in a regular expression. 

The \textquotesingle{}or\textquotesingle{} operation splits the regular expression into two, that get represented by an \mbox{\hyperlink{class_ast}{Ast}} object with two subexpressions.
