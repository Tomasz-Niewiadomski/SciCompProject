\doxysubsection{Detailed Description}
Class for a general abstract syntax tree. 

This class is the base class for all the other \mbox{\hyperlink{class_ast}{Ast}} classes. The virtual identifier functions are subsequentely overridden in the derived classes. The \mbox{\hyperlink{class_ast}{Ast}} class is used to store the regular expression in a tree structure.
